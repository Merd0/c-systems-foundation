# Problem Çözme Örnekleri

Bu klasör, temel syntax öğrendikten sonra küçük senaryolarla C pratiği yapmak
içindir. Amaç sadece doğru sonucu almak değil; input kontrolü, okunabilir akış,
fonksiyonlara bölme ve gerçekçi çıktı üretme alışkanlığı kazanmaktır.

## Önerilen Sıra

| Sıra | Dosya | Odak |
| --- | --- | --- |
| 1 | [`calculator_switch_case.c`](calculator_switch_case.c) | Menü ve `switch` pratiği |
| 2 | [`mini_atm_simulation.c`](mini_atm_simulation.c) | Döngü, menü, hesap durumu |
| 3 | [`password_strength_checker.c`](password_strength_checker.c) | Karakter kontrolü ve koşul tasarımı |
| 4 | [`expense_tracker_summary.c`](expense_tracker_summary.c) | Toplam, ortalama, raporlama |
| 5 | [`inventory_stock_alert.c`](inventory_stock_alert.c) | Stok kontrolü ve uyarı mantığı |
| 6 | [`student_stats_dashboard.c`](student_stats_dashboard.c) | Min/max/ortalama hesaplama |
| 7 | [`sales_report_generator.c`](sales_report_generator.c) | Rapor üretme ve formatlı çıktı |
| 8 | [`user_role_access_simulation.c`](user_role_access_simulation.c) | Rol bazlı erişim mantığı |

## Bitirme Kriteri

Bu klasörü bitirmiş sayılmak için en az iki örneği değiştir:

- Yeni bir input alanı ekle.
- Hatalı input durumunu ele al.
- Çıktıyı daha okunur hale getir.
- Mümkünse bir yardımcı fonksiyon çıkar.
